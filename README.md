# 🎨 منصة تسجيل الفنانين التشكيليين

نظام متكامل لتسجيل الفنانين ورفع أعمالهم الفنية، مع تخزين الصور في جوجل درايف.

## 🚀 المميزات
- تسجيل بيانات الفنانين (الاسم، الكود الجامعي، الهاتف، إلخ)
- رفع صور الأعمال الفنية (حد أقصى 12 صورة)
- تخزين الصور في جوجل درايف في مجلدات بأسماء الفنانين
- واجهة مستخدم عصرية ومتجاوبة

## 🛠️ التقنيات المستخدمة
- HTML + CSS + JavaScript (Tailwind CSS للتنسيق)
- Google Apps Script (خادم الواجهة الخلفية)
- Google Drive API (تخزين الصور)
- Google Sheets API (تسجيل البيانات - اختياري)
